# Miniproject Gigs
Sekumpulan project kecil yang diperoleh dari pekerjaan freelance & remote saat menangani pemecahan masalah cukup kompleks untuk dipecahkan, maupun online challenge dalam mengasah kemampuan coding dan problem solving.<br><br>
Repository kami terdiri atas 3 kategori miniproject untuk saat ini, diantaranya adalah JavaScript, ES6, serta PHP.

## JavaScript Mini Project Portfolio
Sekumpulan mini project javascript dasar yang mayoritas merupakan online challenge, dan dipelajari ulang secara mandiri. <br>

#### Daftar Isi
**1 - Colorflipper**<br>
Merubah warna background sesuai dengan 4 pilihan tombol tersedia, diantaranya adalah Hijau, Merah, Biru, dan Warna Acak.<br>
**2.1 - Stopwatch**<br>
Stopwatch yang di hard coding ulang secara mandiri, menggali fundamental dan prinsip dari JS Dasar.<br>
**2.2 - Stopwatch - 2**<br>
Stopwatch yang bekerja, dari sumber Geeks for Geeks.<br>
**3 - Palindrome Checker**<br>
Menginput suatu kata kunci, lalu mencaritahu kandungan palindrome dari kata yang telah diisikan.<br>
**4 - Quote Generator**<br>
Menghasilkan Quote motivasi secara random dengan menekan tombol 'Generate Quote' secara berulang kali.<br>
**5 - BabelJS Navbar**<br>
Sekumpulan miniproject Navbar dengan tujuan menggali cara menggunakan BabelJS saat berusaha menyatukan Navbar yang awalnya di hard code pada setiap halaman HTML, menjadi Root Navbar tersendiri.<br>
**6 - Pokemon Random Number**<br>
Sebuah project UDEMY Exercise yang menantang diri untuk membuat JavaScript Pokemon Random Generator, sebelum memasuki topik tentang ReactJS.<br>
Cara Main: refresh halaman berkali - kali untuk menghasilkan karakter Pokemon secara random. <br>

## ECMASCRIPT-6 (ES6) Mini Project Portfolio
Sekumpulan mini project ES6, lanjutan dari JavaScript yang sebelumnya pernah dipelajari lewat Bootcamp seputar ES6 Dasar<br><br>

#### Daftar Isi
**1 - Template Literals**<br>
Learning to use the basics of template literals<br>
**2.1 - Destructuring Objects**<br>
Learning on how to use Destructuring Objects<br>
**2.2 - Destructuring Arrays**<br>
Learning how to use destructuring arrays after learning destructuring objects in point 2.1 above.<br>
**3 - Object Literal**<br>
Learn the basicsof Object Literal<br>
**4 - Latihan Ulang Poin diatas No. 1-3**<br>
An exercise that rewire the practical knowledge of point No 1-3 above by using AI to generate exercises that challenge myself<br>
**5 - For of Loop**<br>
Learn the basics of using For of Loop, eventhough we often use this regularly in most real project cases.<br>
**6 - Classes**<br>
Learning to use class extends in ES6, by using the import.<br>
Instructions: run the ES6 JS on a file called playerTennis.js.  You can use 'Run Code' button at the top-right, or right click on the 'playerTennis.js' file, then select Run Code _(but install Code Runner Extension first)_.<br>
**7 - Promises**<br>
Understanding the usage of Promises, before learning about fetch<br>
**8 - Fetch**<br>
Learn how to fetch API<br>
**9 - Async and Await**<br>
Learn and understanding why and when to use the async await, primarily when the promise exceeds its loading time.<br>
